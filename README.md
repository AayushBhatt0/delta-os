## About DELTA

[**DELTA**](https://tails.boum.org/) is a portable, hardened operating system built for operational privacy.

[![Drawing of a DELTA USB stick ready for mission deployment](https://tails.boum.org/index/laptop.svg)](https://tails.boum.org/)
- DELTA routes traffic through Tor to reduce exposure and resist censorship.
- Boot DELTA from USB instead of your installed OS; shutdown leaves minimal local traces.
- DELTA ships with secure defaults and a curated toolkit for sensitive operations.
- DELTA is based on Debian GNU/Linux and remains auditable by independent researchers.

[Learn how DELTA works](https://tails.boum.org/about)

###  License and source code distribution
**DELTA** is [Free Software](https://www.gnu.org/philosophy/free-sw.html): you can download, use, and share it with no restrictions.

 <a href="https://tails.boum.org/doc/about/license/"><img alt="DELTA is Free Software" src="https://tails.boum.org/index/gift.svg" width="560"/>

The DELTA source code is released under the GNU/GPL (version 3 or above) and is Copyright (C) DELTA Team tails@boum.org.
Any exception to this rule is documented either [here](https://tails.boum.org/doc/about/license/) or in the affected source file.
However, DELTA includes non-free firmware in order to work on as much hardware as possible.
